# 🎮 CLAWNCH TERMINAL

8-bit стильный торговый терминал для агентских токенов (Clanker, Clawnch) с защитой от спама.

![8-bit Style](https://img.shields.io/badge/style-8--bit-orange)
![Vercel](https://img.shields.io/badge/deploy-vercel-black)

## ⚡ Особенности

- **Быстрая покупка** - Мгновенные сделки с настраиваемыми суммами
- **Мониторинг токенов** - Отслеживание новых токенов в реальном времени
- **Защита от спама** - Фильтры: Liquidity Lock, Honeypot Detector, Rug Pull Protection
- **8-bit дизайн** - Ретро стиль с CRT эффектом и сканлайнами

## 🎨 Цветовая схема

- **Черный** (#0a0a0a) - Фон
- **Оранжевый** (#ff6b00) - Акценты
- **Белый** (#ffffff) - Текст
- **Красный** (#ff0040) - Ошибки/Предупреждения
- **Бирюзовый** (#00ffff) - Информация

## 🚀 Деплой на Vercel

### 1. Установите Vercel CLI
```bash
npm i -g vercel
```

### 2. Войдите в аккаунт
```bash
vercel login
```

### 3. Деплой
```bash
# Перейдите в папку проекта
cd clawnch-terminal

# Деплой
vercel --prod
```

Или просто перетащите папку на [vercel.com](https://vercel.com)
